WildFly 9.0 Docker File
=========================

Based on the original WildFly Dockerfile - `https://github.com/JBoss-Dockerfiles/wildfly`


Build the Image
---------------

`docker build -t kraluk/wildfly:9.0 .`


Run the Image
-------------

`docker run -it kraluk/wildfly:9.0`
