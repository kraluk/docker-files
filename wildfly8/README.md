WildFly 8.2.1 Docker File
=========================

Based on the original WildFly Dockerfile - `https://github.com/JBoss-Dockerfiles/wildfly`


Build the Image
---------------

`docker build -t kraluk/wildfly:8.2.1 .`


Run the Image
-------------

`docker run -it kraluk/wildfly:8.2.1`
